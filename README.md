# Student Performance Dataset

## Overview

This dataset contains 15 observations related to student performance. It includes study habits, test scores, sleep duration, stress level, and exercise duration. The dataset can be used for basic data analysis, correlation analysis, regression modelling, and machine learning practice.

## Dataset Description

The objective of this dataset is to explore how different lifestyle and study factors may influence a student's test score.

Key variables include:

* Study hours
* Test score
* Sleep hours
* Stress level
* Exercise minutes

| Column       | Description                                                      | Data Type |

## Potential Analysis

This dataset can be used to answer questions such as:

1. Is there a relationship between study hours and test scores?
2. Do students with more sleep hours achieve higher test scores?
3. Does stress level negatively affect test scores?
4. Is exercise duration related to better student performance?
5. Which variable has the strongest relationship with test score?
